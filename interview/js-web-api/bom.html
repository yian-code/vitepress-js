BOM操作（Browser Object Model） #

navigator #

能够检测出用户当前访问网站的 设备、设备的系统、系统的版本、当前的浏览器厂商、浏览器的版本。

  1. navigator 数据可被浏览器使用者更改
  2. 一些浏览器对测试站点会识别错误
  3. 浏览器无法报告晚于浏览器发布的新操作系统
const ua = navigator.userAgent;
const isChrome = ua.indexOf('Chrome');
console.log(isChrome);

location #

属性解释
assign()触发窗口加载并显示指定的URL的内容
hash表示在url路径中以 # 开头 及 #后面的部分内容 (哈希值)
host包含了域名的一个DOMString，可能在该串最后带有一个:并跟上URL的端口号
hostname表示获取url中的域名
href表示获取完整的url路径，获取到的一定是一个绝对路径，设置时可以写相对路径
origin表示获取网络协议 + 域名 + 端口号
pathname表示获取页面所在路径,开头有一个/
port表示获取端口号
protocol包含URL对应协议的一个DOMString，最后有一个:
reload()刷新页面的方法
replace(url)表示在当前页面打开一个新的url页面的方法
search表示获取url中的查询字符串，也就是 ? 及 ?后面的内容，但不包含hash部分：https://www.baidu.com/s?tn=39042058_39_oem_dg&ie=utf-8&wd=MDN

history #

history.length     // 表示当前网站浏览的历史记录条数
history.back()     // 返回上一条历史记录
history.forward()  // 进入下一条历史记录
history.go(n)      // 进入第 n 条记录
