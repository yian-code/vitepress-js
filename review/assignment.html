解构赋值 #

解构赋值的拷贝是浅拷贝，解构赋值拷贝的是这个值的引用，而不是这个值的副本

数组的解构赋值 #

let a = 1;
let b = 2;
let c = 3;

// 解构赋值
let [a, b, c] = [1, 2, 3];

解构赋值的写法属于“模式匹配”，只要等号两边的模式相同，左边的变量就会被赋予对应的值。

let [A, [B], C] = [10, [20], 30];

解构赋值不成功时变量值为undefined（赋值失败呗）

解构赋值可以赋默认值，默认值生效的条件是当一个数组成员严格等于undefined，默认值才会生效。

const [a, b = 2, c] = [1, undefined, 3];

console.log(a, b, c) // 1 2 3

另一种情况是不完全解构，即等号左边的模式，只匹配一部分的等号右边的数组。这种情况下，解构依然可以成功。

let [x, y] = [1, 2, 3];
x // 1
y // 2

let [a, [b], c] = [1, [2, 3], 4];
a // 1
b // 2
c // 4

对象解构赋值 #

当变量名与属性名一致时：

const User = {
    name: 'yian',
    age: 20,
    sex: 1
}
const { name, age, sex } = User;

当变量名不一致时，需要这样做：

const { username: name, age, _sex: sex } = User;

这也说明了对象的解构赋值机制是先找到同名的属性，然后再赋值给对应的变量，真正被赋值的是变量而不是属性

如果变量名与属性名不一致，或者变量声明与赋值分开，则解构失败，值为undefined：

const { Name, age, sex } = User;
console.log(Name, age, sex); // Name is not defined


const { name, age, sex };
{ name, age, sex } = User;
// Uncaught SyntaxError: missing = in destructuring declaration

对象的解构赋值，可以很方便地将现有对象的方法，赋值到某个变量。

// 例一
let { log, sin, cos } = Math;

// 例二
const { log } = console;
log('hello') // hello

解构也可以用于嵌套结构的对象：

let obj = { 
    p: [ 
        'Hello', 
        { 
            y: 'yian' 
        } 
    ] 
};

let { p: [x, { y }] } = obj;
x // "Hello"
y // "yian"

对象的解构也可以指定默认值。默认值生效的条件是，对象的属性值严格等于undefined。：

const {x = 3} = {};
x // 3

const {x = 3} = {x: undefined};
x // 3

const {x = 3} = {x: null};
x // null

当使用已经声明过的变量时：

// 错误的写法  
let x;
{x} = {x: 1};  
// SyntaxError: syntax error

JavaScript 引擎会将{x}理解成一个代码块，从而发生语法错误。只有不将大括号写在行首，避免 JavaScript 将其解释为代码块，才能解决这个问题。

// 正确的写法
let x;
({x} = {x: 1});

字符串的解构赋值 #

字符串也可以解构赋值。这是因为此时，字符串被转换成了一个类似数组的对象。

const [a, b, c, d] = 'yian';
a // y
b // i 
c // a 
d // n

类似数组的对象都有一个length属性，因此还可以对这个属性解构赋值。

let { length : len } = 'yian';
len // 4

函数参数的解构赋值 #

function add([x, y]){
  return x + y;
}

add([1, 2]); // 3

这当我们传入对象的多个属性值作为参数的时候非常有用（函数参数的解构也可以赋默认值）

const User = {
    name: 'yian',
    age: '20',
    _id: '001'
}
function sayHi({ name, age, _id = 0 }) {
    console.log(`${name} + ${age} + ${_id}`)
}

sayHi(User) // yian + 20 + 001

解构赋值的应用 #

  1. 交换变量的值
let x = 1;
let y = 2;

let [x, y] = [y, x];
  1. 函数返回多个值，可以很方便的将多个值分别赋给变量：
// 数组形式
function fn(){ 
    return [1, 2, 3]; 
}
let [a, b, c] = fn();

// 对象形式
function fn(){ 
    return {foo: 1, bar: 2}; 
}
let { foo, boo } = fn();
  1. 函数传递多个参数，并设置默认值
// 以数组形式传递
function fn([x, y, z]) {...}
fn([1, 2, 3]);

// 以对象形式传递
function fn({ x = 0, y = 0, z = 0 }) {...}
fn({ x: 1, y: 2, z: 3 });
  1. 变量Map结构：
const map = new Map()
    .set('a', ['hello', 'world'])
    .set('b', 'yian');

for (const [key, value] of map) {
    console.log(key + ': ' + value);
}
// a: hello,world
// b: yian

for (const [key, ] of map) { }   // 只获取键

for (const [, value] of map) { } // 只获取值
  1. 输入指定模块的指定方法
const { SourceMapConsumer, SourceNode } = require("source-map");
  1. 提取JSON数据：
let jsonData = {
    "id": 42, 
    "status": "ok",
    "data": [12, 2332]
}
let { id, status, data } = jsonData;

console.log(id, status, data) // 42 'ok'  [12, 2332]
