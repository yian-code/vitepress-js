扩展运算符 #

扩展运算符（spread）是三个点（...）。它好比 rest 参数的逆运算，将一个数组转为用逗号分隔的参数序列。

数组拷贝（浅拷贝）

let arr = [1, 2, 3];
let arr2 = [...arr]; // like arr.slice()
arr2.push(4);

// arr2 此时变成 [1, 2, 3, 4]
// arr 不受影响

注意，只有函数调用时，扩展运算符才可以放在圆括号中，否则会报错。

(...[1, 2])
// Uncaught SyntaxError: Unexpected number

console.log((...[1, 2]))
// Uncaught SyntaxError: Unexpected number

console.log(...[1, 2])
// 1 2

遍历数组 #

  1. for：获取数组的下标和元素

  2. for...of：可以用来遍历数组、类数组对象、字符串、Set、Map以及Generator对象

for(let item of arr){ // for-of: 只能拿到值而不能获取下标
	console.log(item); 
}
  1. for...in：
for (let index in arr){
	console.log( typeof index );  // String 
	console.log( arr[index] );
}

尽量不要用for...in来遍历数组

  • for...in遍历的数组键名为字符串类型
  • for...in循环不仅遍历数字键名，还会遍历手动添加的其他键，甚至包括原型链上的键。
  • 某些情况下for...in循环会以任意顺序遍历键名

for...in和for...of区别 #

参见：MDN-for...in和for...of区别

无论是for...in还是for...of语句都是迭代一些东西。它们之间的主要区别在于它们的迭代方式。

  • for...in 语句以任意顺序迭代对象的可枚举属性。

  • for...of 语句遍历可迭代对象定义要迭代的数据。

forEach #

forEach() 方法对数组的每个元素执行一次给定的函数。函数可接收一到三个参数，分别是

  • 数组中正在处理的当前元素。
  • 数组中正在处理的当前元素的索引。
  • forEach() 方法正在操作的数组。
const arr = [1,2,3,4,5];

arr.forEach( (item,index,array) => {
	console.log(item) // 1  2  3  4  5
	console.log(index) // 0  1  2  3  4 
	console.log(array) // Array(5) [ 1, 2, 3, 4, 5 ]
})

entries()、keys()、values() #

三种方法都用于遍历数组，它们都返回一个遍历器对象，可以用for...of遍历

  • keys()：对键名遍历
  • values()：对键值遍历
  • entires()：对键值对遍历
let arr = ['a', 'b', 'c'];

for (let index of arr.keys()) { 
    console.log(index) // 0  1  2
}

for (let val of arr.values()) { 
    console.log(val)   // a  b  c
}

for (let [index, val] of arr.entries()) { 
    console.log([index, val]) 
}
// [ 0, 'a' ]   [ 1, 'b' ] [ 2, 'c' ]

reduce() #

参见：MDN-reduce()

reduce() 方法对数组中的每个元素按序执行一个由您提供的 reducer 函数，每一次运行 reducer 会将先前元素的计算结果作为参数传入，最后将其结果汇总为单个返回值。

arr.reduce(function(pre, curval, index, array){
	...
}, inival);
  • pre：上一次调用函数的返回值，在第一次调用时如果指定了初始值inival，则其值为inival，否则为array[0]
  • curval：数组中正在处理的元素。在第一次调用时，若指定了初始值 inival，其值则为数组索引为 0 的元素 array[0]，否则为 array[1]。
  • index：数组中正在处理的元素的索引。若指定了初始值 inival，则起始索引号为 0，否则从索引 1 起始。
  • array：用于遍历的数组

案例：数组求和

let total = [ 0, 1, 2, 3 ].reduce(
	( previousValue, currentValue ) => previousValue + currentValue
, 0 );
// total =  6

concat()：合并 #

合并两个或多个数组，不会改变原来的数组，返回一个新的数组（纯函数）

const a = [1, 2];
const b = [3, 4];

let c = a.concat(b); // Array(4) [ 1, 2, 3, 4 ]

可以使用扩展运算符来代替

let d = [...a, ...b]; // Array(4) [ 1, 2, 3, 4 ]

reverse()反转数组 #

reverse()方法将数组元素位置颠倒，该方法会改变原数组

const a = [1, 2, 3, 4];

let b = a.reverse()

console.log(a) // Array(4) [ 4, 3, 2, 1 ]
console.log(b) // Array(4) [ 4, 3, 2, 1 ]

经典反转字符串：

let rts = str.split('').reverse().join('');

slice()：切片 #

arr.slice([begin[, end]])：数组切片，返回一个新的数组（纯函数）

  • begin：提取开始位置

    • 为负数则从原数组中倒数第几个开始取
    • 省略它则从0开始取
    • 大于原数组长度则返回空数组

  • end：提取结束位置

    • 为负数则从倒数第几个结束提取（不包含最后一个元素）
    • 省略它则会一直提取到数组末尾
    • 大于数组的长度也会一直提取到数组末尾
// const arr = ["a1","a2",'a3',"a4","a5"];

let val1 = arr.slice(1,3);   // [ "a2", "a3" ]
let val2 = arr.slice(-3);    // [ "a3", "a4", "a5" ]
let val3 = arr.slice(3);     // [ "a4", "a5" ]
let val4 = arr.slice();      // [ "a1", "a2", "a3", "a4", "a5" ]
let val5 = arr.slice(-3,-1); // [ "a3", "a4" ]
let val6 = arr.slice(5);     // [  ]

splice()：剪接 #

array.splice(start[, deleteCount[, item1[, item2[, ...]]]])：

  • start：指定修改的位置
  • deleteCount：要移出的数组元素个数
    • 大于start后元素的个数总数，则一直移出到数组末尾
    • 省略则start之后数组的所有元素都会被删除
    • deleteCount 是 0 或者负数，则不移除元素。这种情况下，至少应添加一个新元素。
    • item1, item2...：要添加进数组的元素，从start 位置开始。如果不指定，则 splice() 将只删除数组元素。

通过删除或替换现有元素或者原地添加新的元素来修改数组，并以数组形式返回被修改的内容，此方法会改变原数组。

splice的返回值：由被删除的元素组成的一个数组

  • 如果只删除了一个元素，则返回只包含一个元素的数组
  • 如果没有删除元素，则返回空数组。

添加元素：

let arr = [1, 2, 3, 4];

let a = arr.splice(2, 0, 'hello');

console.log('arr: ', arr) // arr: [ 1, 2, "hello", 3, 4 ]
console.log('a', a); // a: [  ]

修改元素：

let arr = [1, 2, 3, 4];

let a = arr.splice(2, 1, 'hello');

console.log('arr: ', arr) // arr: [ 1, 2, "hello", 4 ]
console.log('a', a); // a: [ 3 ]

删除元素：

let arr = [1, 2, 3, 4];

let a = arr.splice(2, 1);

console.log('arr: ', arr) // arr: [ 1, 2, 4 ]
console.log('a', a); // a: [ 3 ]

indexOf()和lastIndexOf() #

  • indexOf()：从前向后查找一个元素在数组中的第一个索引，如果不存在则返回-1
  • lastIndexOf：从后向前查找一个元素在数组中的最后一个索引，如果不存在则返回-1
let arr = [1, 2, 2, 2];

let index = arr.indexOf(2); // 1
let lastindex = arr.lastIndexOf(2); // 3
let err = arr.lastIndexOf(9); // -1

存在判断：includes() #

const arr = ['1', '2'];
arr.includes('1'); // true

find()和findIndex()（搜索单个元素） #

find()找到数组中符合特定条件的对象，findIndex()返回找到元素的索引，而不是元素本身。并且在未找到任何内容时返回 -1。

let result = arr.find(function(item, index, array) {
  // 如果返回 true，则返回 item 并停止迭代
  // 对于假值（falsy）的情况，则返回 undefined
});
  • item 是元素。
  • index 是它的索引。
  • array 是数组本身。

如果它返回 true，则搜索停止，并返回 item。如果没有搜索到，则返回 undefined。

let users = [
  {id: 1, name: "John"},
  {id: 2, name: "Pete"},
  {id: 3, name: "Mary"}
];

let user = users.find(item => item.id == 1);

alert(user.name); // John

map()和filter()过滤器 #

  • map()：返回一个全新的数组，这个新数组由原数组中的每个元素都调用一次callback函数后的返回值组成。

  • filter()：过滤器，返回通过测试的元素组成的数组，没有任何元素通过则返回空数组

  • callback回调函数有三个参数分别是：

  1. element：当前正在处理的数组元素。
  2. index：正在处理的数组元素下标。
  3. array：map()（filter()）被调用的数组。
  • thisArg：执行callback时使用的this值
const arr = [1, 2, 3, 4];

let d = arr.map( (ele) => {
    return ele * 3;
})
console.log(d) // [ 3, 6, 9, 12 ]

let e = arr.filter( ele => { return ele > 2 } )
console.log(e) // [ 3, 4 ]

join()、数组转字符串 #

join()：将所有的数组元素被转换成字符串，再用一个分隔符将这些字符串连接起来。如果数组只有一个项目，那么将返回该项目而不使用分隔符。不会改变原数组

let arr = [1, 2, 2, 2];

let str2 = arr.join('') 

console.log(str2) // '1222'
console.log(arr)  // [ 1, 2, 2, 2 ]
  • String()
  • toString()
let arr = [1, 2, 2, 2];

let str = arr.toString();

console.log(str) // '1,2,2,2'
console.log(arr) // [ 1, 2, 2, 2 ]

let str1 = String(arr)

console.log(str1) // '1,2,2,2'
console.log(arr)  // [ 1, 2, 2, 2 ]

some()和every() #

  • some()：查询数组中是否存在满足某些条件的数组项,满足则返回true 不满足返回false

  • every()：查询数组项中的所有数组项是否全部满足条件，如果全都满足返回true只要有一个不满足 返回的就是false

两个函数的参数都相同，语法相近，以some()为例：arr.some(callback(element[, index[, array]])[, thisArg])

  • callback回调函数有三个参数分别是：
  1. element：当前正在处理的数组元素。
  2. index：正在处理的数组元素下标。
  3. array：some()被调用的数组。
  • thisArg：执行callback时使用的this值
const arr = [1, 2, 3, 4];

let a = arr.some( (ele) => {
    return ele > 3;
})
console.log(a) // true

let b = arr.every( (ele) => {
    return ele > 3;
})
console.log(b) // false

sort()数组排序（升序降序） #

sort()方法用原地算法对数组的元素进行排序，并返回数组。默认排序顺序是在将元素转换为字符串，然后比较它们的UTF-16代码单元值序列时构建的。

let arr = [1,22,11,2,4]

arr.sort( (a,b) => { return a - b })
console.log(arr);// [ 1, 2, 4, 11, 22 ]

arr.sort( (a,b) => { return b - a })
console.log(arr); // [ 22, 11, 4, 2, 1 ]

from()：转为真正的数组、of() #

from()用于将两类对象转为真正的数组：

  • 类似数组的对象（array-like object）
  • 可遍历（iterable）的对象（包括 ES6 新增的数据结构 Set 和 Map）。
// NodeList 对象
const btns = document.querySelectorAll('button');
Array.from(btns).filter(item => { return item.textContent.length > 100; });

// arguments 对象
function foo() {
  let args = Array.from(arguments);
  // ...
}

of()：将一组值转为数组：

Array.of(1, 2, 3) // [1, 2, 3]
Array.of()        // []

fill()：填充数组 #

使用给定值填充数组，fill方法还可以接受第二个和第三个参数，用于指定填充的起始位置和结束位置

['a', 'b', '3'].fill(6)   // [ 6, 6, 6 ]

[1, 2, 3, 4, 5, 6, 7, 8, 9, 0].fill('a', 3, 8);
// [1, 2, 3, 'a', 'a', 'a', 'a', 'a', 9, 0]

flat()、flatMap() #

flat()：将数组降维

  • 默认拉平一层，你可以传一个整数参数表示想要拉平的维度
  • 如果将N维数组转为一维数组，则参数给Infinity
  • 原数组有空位则会跳过空位
[1, 2, [3, [4, 5]]].flat(2)  // [1, 2, 3, 4, 5]

[1, [2, [3]]].flat(Infinity) // [1, 2, 3]

[1, 2, , 4, 5].flat() // [1, 2, 4, 5]

flatMap()：相当于对原数组每个成员执行一次map()函数再执行flat()方法展开，该方法返回一个数组，不改变原数组

[1, 2, 3, 4].flatMap(x => x * 2) // [ 2, 4, 6, 8 ]

// 只能展开一层数组
[ [1], [2], [3], [4] ].flatMap(x => x * 2) // [ 2, 4, 6, 8 ]

[ 1, 2, [ 3, [4] ] ].flatMap(x => x * 2)  // [ 2, 4, NaN ]

at() #

接受一个整数作为参数，返回对应位置的成员，并支持负索引。这个方法不仅可用于数组，也可用于字符串和类型数组（TypedArray）。

const arr = [1, 3, 21, 44];
arr.at(2)                   // 21
arr.at(-2)                  // 21
arr.at(4)                   // undefined
