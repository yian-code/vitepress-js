正则表达式中的特殊字符 #

表达式描述
[abc]查找方括号之间的任何字符，包括转义序列，可以使用破折号-来指定一个字符范围。例如,[abcd]和[a-d]是一样的，他们都匹配"brisket"中的‘b’，也都匹配“city”中的‘c’
[^abc]一个反向字符集，它匹配任何没有包含在方括号中的字符
[0-9]查找任何从0至9的数字，[]中只匹配一个，匹配多个需要加{}字符来限制，例如[0-9]{1,3}
x|y匹配x或者y ，等价于[xy]
()小括号()中的内容被作为单个字符来匹配
{n}n是一个正整数，匹配了前面一个字符刚好出现了n次
{n,}n是一个正整数，匹配前一个字符至少出现了n次
{n,m}n和m都是整数。匹配前面的字符至少n次，最多m次，如果n或者m的值是0，这个值被忽略
^匹配输入的开始.如果多行标志被设置为 true，那么也匹配换行符后紧跟的位置。例如：/^A/并不会匹配"an A"中的'A'，但是会匹配"An E"中的'A'
$匹配输入的结束.如果多行标志被设置为 true，那么也匹配换行符前的位置。例如：/t$/并不会匹配"eater"中的't'，但是会匹配"eat" 中的 't'
.默认匹配除换行符之外的任何单个字符
n+匹配前面一个表达式 1 次或者多次.等价于 {1,}
n*匹配前一个表达式 0 次或多次.等价于{0,}
n?匹配前面一个表达式 0 次或者 1 次.等价于 {0,1}
\d匹配一个数字.等价于[0-9]
\D匹配一个非数字字符.等价于[^0-9]
\w匹配一个单字字符(字母,数字或者下划线)，等价于 [A-Za-z0-9_]
\W匹配一个非单字字符.等价于 [^A-Za-z0-9_]
\s查找空白字符，包括空格、制表符、换页符和换行符等价于：[ \f\n\r\t\v\u00a0\u1680\u180e\u2000-\u200a\u2028\u2029\u202f\u205f\u3000\ufeff]
\S等价于[ \f\n\r\t\v\u00a0\u1680\u180e\u2000-\u200a\u2028\u2029\u202f\u205f\u3000\ufeff]
\b匹配单词边界 /\bm/匹配moon中的m
\B匹配一个非单词边界
\uxx匹配一个两位十六进制数(\x00-\xFF)表示的字符
\uxxxx查找以十六进制数xxxx规定的Unicode字符

正则表达式标志 #

修饰符描述
i执行对大小写不敏感的匹配
g执行全局匹配(查找所有匹配并非找到第一个就停止)
m执行多行匹配
s允许 . 匹配换行符
u使用unicode码的模式进行匹配
y执行“粘性(sticky)”搜索，匹配从目标字符串的当前位置开始

exec #

RegExp/exec：一个在字符串中执行查找匹配的RegExp方法，它返回一个数组（包含额外的属性index和input，未匹配到则返回null）。默认会将正则整体匹配到的内容作为数组的第一项，如果正则中有分组()，那么会将每一个分组中匹配到的内容作为数组的后几项来返回

let str = 'wjeqwkjeqwqnkjqwnwqqeqe';
let reg = /eq/g;

let execArr = reg.exec(str)
console.log(execArr)
// Array [ "eq" ]​
// 0: "eq"
// groups: undefined
// index: 2   --匹配到的字符位于原始字符串的基于0的索引值
// input: "wjeqwkjeqwqnkjqwnwqqeqe"   --原始字符串
// length: 1
// <prototype>: Array []

test #

一个在字符串中测试是否匹配的RegExp方法，它返回true或false。

let str = 'wjeqwk';
let reg = /z/g;

let val = reg.test(str)
console.log(val)  // false

match #

str.match(regexp)。使用g标志时，match方法会返回与完整正则表达式匹配的所有结果，但不会返回捕获组，如果没有匹配到则返回null。

let date = '2022-5-21'
let reg = /[0-9]+/g;

let valArr = date.match(reg)
console.log(valArr) // [ "2022", "5", "21" ]

matchAll #

一个在字符串中执行查找所有匹配的String方法，它返回一个迭代器(参考ES6-iterator)。

let date = '2022-5-21'
let reg = /[0-9]+/g;
let result = date.matchAll(reg)

console.log(result)
// RegExp String Iterator {  }
// <prototype>: RegExp String Iterator { … }

for (let val of result) { 
    console.log(val)
}
// Array [ "2022" ] 
// Array [ "5" ]
// Array [ "21" ]

search #

一个在字符串中测试匹配的String方法，它返回该正则模式的第一个匹配项的在字符串中的位置索引，或者在失败时返回-1。

let date = '2022-5-21'
let reg = /-/g;

let val = date.search(reg)
console.log(val) // 4

replace #

regexp[Symbol.replace](str, newSubStr|function)：一个在字符串中执行查找匹配的String方法，并且使用替换字符串替换所有符合正则模式的匹配项，并返回替换后的新字符串结果。用来替换的参数可以是一个字符串或是一个针对每次匹配的回调函数。

let date = '2022-5-21'
let reg = /-/g;

let v = date.replace(reg,":")
console.log(v) // 2022:5:21

split #

一个使用正则表达式或者一个固定字符串分隔一个字符串，并将分隔后的子字符串存储到数组中的String方法

let reg = /-/g;
let str = '2016-01-02';

let result = reg[Symbol.split](str);
console.log(result); // ["2016", "01", "02"]
